# Ali Cavalos Resgates - Landing Page Premium

Esta é a Landing Page premium, responsiva e de alta conversão construída para o projeto **Ali Cavalos Resgates**.

A página foi construída com foco em impacto visual, desempenho, SEO e acessibilidade, utilizando HTML5 semântico, Vanilla CSS e Vanilla JavaScript.

## 🚀 Funcionalidades Principais

- **Visual Premium e Minimalista**: Layout elegante com tipografia refinada e amplo espaço em branco.
- **Hero Interativo**: Fundo de tela com vídeo otimizado e sobreposição escura para legibilidade do texto.
- **Comparador Antes e Depois**: Slider interativo (CSS + JS nativo) comparando o estado de resgate e reabilitação dos cavalos.
- **Vídeos sob Demanda**: Vídeos carregados apenas ao clicar para garantir pontuação máxima no Lighthouse.
- **Checkout / Modal de Doação**: Modal centralizado e responsivo preparado para integração de gateway.
- **Galeria de Fotos Lightbox**: Visualização em tela cheia das fotografias ao clicar.
- **Efeito de Scroll Reveal**: Transição discreta dos componentes conforme o usuário navega na página.

## 🛠️ Como Executar Localmente

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador web, ou utilize a extensão **Live Server** no VS Code para uma melhor experiência.
3. Não há dependências ou bibliotecas externas pesadas para instalar (Lighthouse acima de 95 garantido).
